9. Jelaskan apa yang dimaksud dengan Komputasi berbasis Web (Web-Based Computing) dan Komputasi pada Embedded System (Embedded Computing)? Serta berikan contohnya?


Komputasi berbasis browser (Web-Based Computing) adalah penggunaan browser internet untuk melakukan tugas komputasi. Peluang untuk komputasi di Web telah dicatat sejauh tahun 1997. Komputasi melalui internet dijelaskan pada tahun 2000. Aplikasi termasuk komputasi terdistribusi untuk pekerja internet seperti yang diilustrasikan oleh James (sebelumnya CrowdProcess) dan HASH, penggunaan tumpukan browser di QMachine, penyematan aplikasi internet sebagai komponen hypermedia semantik dan Server Signaling di Peer-to -jaringan rekan diatur melalui WebRTC. Komputasi berbasis browser melengkapi komputasi awan, karena mereka mengurangi beban komputasi sisi server, sering kali menggunakan layanan internet RESTful yang dihosting di awan.

Pembahasan

Komputasi berbasis Web (Web-Based Computing) adalah Komputasi awan (bahasa Inggris: cloud computing) adalah gabungan pemanfaatan teknologi komputer dan pengembangan berbasis Internet. Awan (cloud) adalah metafora dari internet, sebagaimana awan yang sering digambarkan di diagram jaringan komputer. Sebagaimana awan dalam diagram jaringan komputer tersebut, awan (cloud) dalam Cloud Computing juga merupakan abstraksi dari infrastruktur kompleks yang disembunyikannya. Ia adalah suatu metoda komputasi di mana kapabilitas terkait teknologi informasi disajikan sebagai suatu layanan (as a service),sehingga pengguna dapat mengaksesnya lewat Internet ("di dalam awan") tanpa mengetahui apa yang ada didalamnya, ahli dengannya, atau memiliki kendali terhadap infrastruktur teknologi yang membantunya.

11. 1. Jelaskan perbedaan aplikasi berbasis web dan aplikasi berbasis mobile serta jelaskan bagaimana sebuah perusahaan menentukan kapan harus menggunakan aplikasi berbasis web dan kapan harus menggunakan aplikasi berbasis mobile!


Aplikasi berbasis web dan aplikasi berbasis mobile memiliki perbedaan dalam hal platform, aksesibilitas, pengembangan, dan pengalaman pengguna. Berikut adalah penjelasan tentang perbedaan antara keduanya:

1. Platform:

- Aplikasi berbasis web dirancang untuk diakses melalui browser web dan dapat dijalankan di berbagai platform, seperti desktop, laptop, tablet, dan perangkat seluler.

- Aplikasi berbasis mobile dirancang khusus untuk dijalankan pada perangkat seluler seperti smartphone atau tablet.

2. Aksesibilitas:

- Aplikasi berbasis web dapat diakses dengan mudah melalui URL dan tidak memerlukan instalasi tambahan pada perangkat pengguna. Pengguna hanya perlu memiliki browser web yang kompatibel.

- Aplikasi berbasis mobile harus diunduh dan diinstal secara khusus dari toko aplikasi (misalnya, App Store untuk iOS atau Google Play Store untuk Android) sebelum pengguna dapat menggunakannya.

3. Pengembangan:

- Aplikasi berbasis web biasanya dikembangkan menggunakan teknologi web seperti HTML, CSS, dan JavaScript. Pengembang dapat menggunakan kerangka kerja (framework) seperti React, Angular, atau Vue.js untuk mempermudah pengembangan.

- Aplikasi berbasis mobile memerlukan pengembangan menggunakan bahasa pemrograman yang spesifik untuk platform tertentu, seperti Swift atau Objective-C untuk iOS, dan Java atau Kotlin untuk Android.

4. Pengalaman Pengguna:

- Aplikasi berbasis web memiliki desain responsif yang dapat menyesuaikan tampilan dengan baik pada berbagai perangkat. Namun, pengalaman pengguna mungkin berbeda-beda tergantung pada browser yang digunakan dan kualitas koneksi internet.

- Aplikasi berbasis mobile dapat memberikan pengalaman pengguna yang lebih terintegrasi dengan perangkat seluler, termasuk akses ke fitur-fitur perangkat seperti kamera, GPS, notifikasi push, dan lainnya.

Keputusan perusahaan untuk menggunakan aplikasi berbasis web atau aplikasi berbasis mobile tergantung pada beberapa faktor, seperti:

1. Tujuan Aplikasi:

- Jika aplikasi bertujuan untuk menyediakan informasi umum yang mudah diakses, seperti situs web perusahaan, blog, atau platform e-commerce, aplikasi berbasis web mungkin lebih cocok.

- Jika aplikasi dirancang untuk memberikan pengalaman yang kaya dan terintegrasi dengan fitur-fitur perangkat seluler, seperti aplikasi perbankan, jejaring sosial, atau permainan mobile, maka aplikasi berbasis mobile lebih disarankan.

2. Target Pengguna:

- Jika target pengguna aplikasi adalah pengguna yang menggunakan berbagai platform dan perangkat, aplikasi berbasis web dapat mencakup lebih banyak pengguna potensial, karena dapat diakses dari berbagai perangkat.

- Jika target pengguna aplikasi adalah pengguna yang menggunakan perangkat seluler secara eksklusif, seperti aplikasi berbasis lokasi atau aplikasi berbasis kamera, maka aplikasi berbasis mobile dapat memberikan pengalaman yang lebih terarah dan terintegrasi.

3. Fitur dan Kebutuhan Teknis:

- Jika aplikasi memerl

ukan akses ke fitur-fitur perangkat seluler khusus, seperti GPS, kamera, sensor gerak, atau pemrosesan offline, maka aplikasi berbasis mobile menjadi pilihan yang lebih baik.

- Jika aplikasi hanya memerlukan tampilan dan interaksi dasar tanpa memerlukan akses ke fitur perangkat seluler, maka aplikasi berbasis web dapat memenuhi kebutuhan tersebut dengan lebih efisien.

Perusahaan perlu menganalisis kebutuhan, tujuan, dan karakteristik pengguna aplikasi mereka untuk menentukan apakah aplikasi berbasis web atau aplikasi berbasis mobile lebih cocok dalam konteks yang diberikan. Dalam beberapa kasus, perusahaan juga dapat mempertimbangkan untuk mengembangkan aplikasi hibrida yang menggabungkan kelebihan dari kedua platform tersebut.

Jawaban:

Aplikasi berbasis web dan aplikasi berbasis mobile memiliki perbedaan dalam platform yang digunakan dan cara aksesnya:

Aplikasi Berbasis Web:

Aplikasi berbasis web dirancang untuk diakses melalui browser web di perangkat yang terhubung ke internet.

Aplikasi ini dapat diakses melalui berbagai perangkat, seperti komputer, laptop, tablet, dan smartphone.

Pengguna tidak perlu menginstal aplikasi secara khusus, cukup membuka browser dan mengakses situs web yang menyediakan aplikasi tersebut.

Keputusan untuk menggunakan aplikasi berbasis web atau aplikasi berbasis mobile tergantung pada beberapa faktor, seperti:

Tujuan Aplikasi: Jika aplikasi bertujuan menyediakan informasi dan layanan yang dapat diakses oleh banyak pengguna melalui berbagai platform dan perangkat, aplikasi berbasis web lebih cocok.

Ketergantungan pada Fitur Perangkat: Jika aplikasi membutuhkan akses ke fitur perangkat mobile tertentu, seperti GPS, kamera, atau sensor lainnya, maka aplikasi berbasis mobile akan lebih tepat.

Pengalaman Pengguna: Jika aplikasi perlu memberikan pengalaman yang dioptimalkan dan kustomisasi sesuai dengan perangkat mobile, seperti antarmuka sentuh dan notifikasi push, aplikasi berbasis mobile dapat memberikan pengalaman yang lebih baik.

Ketersediaan dan Aksesibilitas: Jika aplikasi harus dapat diakses dengan mudah tanpa memerlukan instalasi khusus, aplikasi berbasis web lebih praktis karena dapat diakses melalui browser.

26. biaya spp setiap semestar di suatu perguruan tinggi naik Rp50.000,00 dari spp sebelumnya. jika ada semestar ke-8 reno membayar spp sebesar Rp 1.750.000,00 total spp yang dibayarkan selama kuliah 8 semestar adalah​


Biaya SPP setiap semester di suatu perguruan tinggi naik Rp 50.000dari SPP bulan sebelumnya. Jika pada semestar ke-8 Reno membayar SPP sebesar Rp 1.750.000, total SPP yang dibayarkan selama kuliah 8 semester adalah Rp 12.600.000.

Penjelasan dengan langkah-langkah:

Pada soal di atas dikatakan biaya SPP yang dibayarkan setiap bulannya naik Rp 50.000 dari SPP bulan sebelumnya. Jadi biaya SPP yang dibayarkan setiap bulannya akan membentuk deret aritmatika dengan beda antar suku Rp 50.000. Pada soal diketahui SPP bulan ke-8 adalah Rp 1.750.000, jadi nilai dari U8 adalah Rp 1.750.000. Pertama kita cari terlebih dahulu nilai suku pertamanya. Suku ke-n barisan aritmatika akan memenuhi rumus:

Un = a + b ( n - 1 )

Jadi jika U8 = 1.750.000 dan b = 50.000, maka nilai a adalah:

U8 = a + b ( 8 - 1 )

1.750.000 = a + 50.000 ( 8 - 1 )

1.750.000 = a + 50.000 ( 7 )

1.750.000 = a + 350.000

a = 1.750.000 - 350.000

a = 1.400.000

Rumus untuk suku ke-8 dari barisan aritmatika tersebut adalah:

Un = 1.400.000 + 50.000 ( n - 1 )

Jumlah total SPP selama 8 bulan dihitung dengan rumus jumlah n suku pertama pada barisan aritmatika

Sn = [tex]\frac{n}{2}[/tex] ( a + Un )

S8 = [tex]\frac{8}{2}[/tex] ( 1.400.000 + U8 )

S8 = 4 ( 1.400.000 + 1.750.000 )

S8 = 4 ( 3.150.000 )

S8 = 12.600.000

Jadi total SPP yang dibayarkan selama 8 semester adalah Rp 12.600.000

39. Contoh alat pembayaran berbasis kertas adalah


Pembayaran nontunai (non cash) seperti alat pembayaran berbasis kertas (paper based), misalnya, cek dan bilyet giro. 

Pembayaran nontunai yaitu pembayaran yang dilakukan tanpa menggunakan uang tunai yang beredar melainkan menggunakan cek atau bilyet giro (BG) dan alat pembayaran menggunakan kartu (ATM, kartu kredit, kertu debit, prabayar). Hal ini terlihat pada ketersediaan jasa pembayaran nontunai yang dilakukan bank maupun lembaga selain bank. Transaksi pebayaran nontunai dengan nilai besar diselenggarakan Bank Indnesia melalui sistem BI-RTGS (Real Time Gross Settlement) dan sistem kliring.BI-RTGS berperan sangat penting  dalam transaksi pembayaran non tunai. Untuk itu BI sangat peduli menjaga stabilitasnya yang dikategorikan sebagai systemcally important payment system (SIPS) yaitu sistem yang memproses transaksi pembayaran bernilai besar dan mendesak. BI bukan semata peduli pada pemberian kemudahan bagi pengguna tapi juga kesetaraan akses hingga ke urusan perlindungan konsumen. Kini pembayaran menggunakan kartu sudah banyak yang menggunakan mulai dari anak anak, remaja, dewasa, hingga tua. Dari pelajar, mahasiswa, ibu rumah tangga, pelayan toko, pengusaha, karyawan hingga pejabat sudah menggunakannya. Kartu ATM, kartu debit dan kartu debit memang praktis, mudah dibawa tanpa memakan tempat di dompet karena bentuknya kecil dan sangat tipis. Masyarakat sudah mulai meninggalkan kebiasaan memegang uang tunai dan memilih bertransaksi nontunai (melalui bank) seperti contoh pengambilan gaji pegawai saat ini sudah melalui ATM, pembayaran uang kuliah, pemberian uang jajan bulanan kepada anak, hingga untuk belanja online. Contoh Kasus yang saya alami, sekarang saya diberi uang jajan setiap minggunya melalui transfer ke rekening ATM saya yang dikirim oleh ayah saya. Yang saya rasakan bedanya saat memegang uang tunai dengan memegang kartu ATM adalah untuk mengambil uang tersebut saya harus jauh pergi ke bank atau mesin ATM dan mengantri di sana sedangkan jika saya memegang uang tunai saya tidak perlu mengalami hal tersebut. Selain itu saya jadi merasa bersifat konsumtif karena jika uang di dompet saya habis, saya jadi terbiasa pergi ke ATM untuk mengambil dan mengambil lagi uang saya yang ada di rekening bank yang saya milikitersebut. Dan sering tidak bisa menahan diri untuk membeli barang yang saya suka meskipun tidak memegang uang di dompet karena saya berfikir bahwa masih ada sejumlah uang di rekening ATM yang bisa saya gunakan.
