Perbedaan utama antara kedua adalah isi kontennya. Jika Web Page hanya menyediakan informasi tentang suatu topik, Website menawarkan banyak informasi tentang topik (Website ini difokuskan pada topik-nya yang biasanya terjadi).

Web Page
Sebuah halaman web atau web page adalah sebuah dokumen atau informasi sumber daya yang cocok untuk World Wide Web dan dapat diakses melalui web browser dan ditampilkan pada monitor atau perangkat mobile.
Contoh: https://en.wikipedia.org/wiki/Earth

Website
Sebuah situs web (sering pula disingkat menjadi situs saja; web site, site) adalah sebutan bagi sekelompok halaman web (web page), yang umumnya merupakan bagian dari suatu nama domain (domain name) atau subdomain di World Wide Web (WWW) di Internet maupun. WWW terdiri dari seluruh situs web yang tersedia kepada publik. 
Contoh: https://www.wikipedia.org

14. SOAL PRODUKTIFI KELAS XI TKJ (A-D) 1. Jelaskan yang di maksud dengan web server, dan berikan 2 contoh serta jelaskan web server yang banyak digunakan pada saat ini! 2. Sebutkan 3 contoh aplikasi Web Browser dan jelaskan masing-masing 3 contoh? 3. sebutkan 3 contoh bahasa pemrograman yang digunakan untuk membuat sebuah web. Kemudian jelaskan cara kerja sebuah web server! 4. Jelaskan apa yang dimaksud dengan DNS Server! 5. Sebutkan dan jelaskan fungsi dari DNS Server!​


Jawaban:

1. Web server adalah sebuah program atau software yang berfungsi untuk menerima dan memproses permintaan dari browser web untuk menampilkan halaman web.

Dua contoh web server yang banyak digunakan saat ini adalah Apache dan Nginx. Apache adalah web server open-source yang sangat populer dan digunakan oleh sebagian besar situs web di seluruh dunia. Nginx adalah web server open-source yang dirancang untuk menangani lalu lintas web dengan efisien pada server modern yang sibuk.

2. Tiga contoh aplikasi web browser yang populer adalah:

Google Chrome: aplikasi web browser yang dikembangkan oleh Google. Fiturnya meliputi tab browsing, penjelajahan pribadi, dan integrasi dengan Google Drive dan aplikasi Google lainnya.Mozilla Firefox: aplikasi web browser open-source yang populer dengan fitur-fitur seperti tab browsing, penjelajahan pribadi, dan dukungan untuk add-on dan ekstensi.Microsoft Edge: aplikasi web browser yang dikembangkan oleh Microsoft. Fiturnya meliputi tab browsing, integrasi dengan Microsoft Office dan OneDrive, dan mode penghematan baterai untuk perangkat mobile.

3. Tiga contoh bahasa pemrograman yang digunakan untuk membuat web adalah:

HTML (Hypertext Markup Language): bahasa markah yang digunakan untuk membuat struktur dan konten halaman web.CSS (Cascading Style Sheets): bahasa stylesheet yang digunakan untuk mengatur tampilan dan format halaman web.JavaScript: bahasa scripting yang digunakan untuk membuat interaksi dan efek di halaman web.

Web server bekerja dengan menerima permintaan dari browser web melalui protokol HTTP (Hypertext Transfer Protocol) atau HTTPS (Hypertext Transfer Protocol Secure). Setelah menerima permintaan, server memproses permintaan tersebut dan mengirimkan kembali halaman web yang diminta ke browser.

4. DNS Server (Domain Name System Server) adalah sebuah server yang berfungsi untuk mengubah nama domain (seperti google.com) menjadi alamat IP yang sesuai (seperti 172.217.168.46) sehingga dapat diakses oleh komputer melalui jaringan internet.

5. Fungsi dari DNS Server adalah sebagai berikut:

Menerjemahkan nama domain menjadi alamat IP yang sesuai.Mengelola cache DNS untuk meningkatkan kecepatan akses ke situs web yang sering dikunjungi.Mencegah serangan DNS Spoofing atau DNS Cache Poisoning dengan memastikan bahwa alamat IP yang dikembalikan benar-benar berasal dari server DNS yang sah.

30. Bagaimana cara kerja laravel?


Berikut penjelasan singkat MVC:
Model yang dimaksud dalam MVC adalah sekumpulan object yang berfungsi untuk mengelola request query ke database di aplikasi.
View adalah sekumpulan object yang berfungsi untuk mengelola atau menampilkan antarmuka ke user aplikasi
Controller berfungsi untuk menangani logika program serta menghubungkan antara Model dengan View.

Berikut gambarannya jika suatu request MVC di ilustrasikan:



Penjelasan dari gambar di atas:
User melakukan inputan request melalui browser atau form.
Request tersebut kemudian akan di terima controller untuk diproses requestnya.
Dianggap request tersebut melibatkan request ke database, maka diteruskanlah request tersebut ke Model. Model kemudianmengambil data yang diminta controller.
Setelah mendapatkan data yang diminta user, maka akan dikembalikan ke controller.
Kemudian Controller akan meneruskan data yang diberikan model ke View.
View kemudian memberikan data kepada user sesuai request.
Sedangkan jika diimplementasikan ke dalam struktur aplikasi Laravel 5.4 maka MVC tersebut terdiri dari:
Model: Seluruh file-file yang berfungsi sebagai model disimpan di dalam folder App/
View: Seluruh file-file yang berfungsi sebagai View berada di dalam folder resources/views/
Controller: Fungsi Controller ditaruh di dalam folder app/Http/Controllers/ dan file web.php yang terletak di folder routes/

31. cara kerja web hosting dan penggunaan aplikasi penggunaan web


Cara kerja web hosting adalah sebagai berikut web hosting sendiri merupakan sebuah komputer yang berfungsi sebagai server yang bertugas sebagai tempat untuk menyimpan atau menaruh file-file sebuah website. File-file inilah yang sebenarnya kita akses dari komputer kita atau perangkat lainnya seperti smartphone, laptop dan sebagainya melalui jaringan internet. Untuk kemudahan akses suatu website kita biasanya mengakses website dengan mengetikan nama  domainnya.

Untuk penggunaan aplikasi dalam website web hosting sendiri seperti yang sudah dijelaskan diatas adalah sebagai tempat untuk penyimpanan file-file suatu website yang nantikan akan diakses oleh pengguna dengan aplikasi web browser. Macam-macam web browser sendiri banyak jenisnya, seperti Internet Explore, Mozilla Firefox dan Google Chrome.

Jadi kesimpulannya adalah web hosting sebagai tempat peyimpanan file-file suatu website. Demikian penjelasan mengenai cara kerja web hsoting dan penggunaannya, semoga bisa membantu.
