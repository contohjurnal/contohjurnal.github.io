1. Jika q(x)=x² ; h(x)=√x-2 maka (hoq)(x) adalah


Untuk menghitung (hoq)(x), kita perlu menggantikan x dalam h(x) dengan q(x).

Dalam hal ini, q(x) = x² dan h(x) = √x-2.

Maka (hoq)(x) = h(q(x)) = h(x²).

Gantikan x dalam h(x²) dengan x²:

(hoq)(x) = h(x²) = √(x²) - 2

Sementara √(x²) adalah x untuk x ≥ 0, kita bisa menggantikannya:

(hoq)(x) = h(x²) = x - 2

Jadi, (hoq)(x) adalah x - 2.
