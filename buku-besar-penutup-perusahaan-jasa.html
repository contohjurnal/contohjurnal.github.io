SHU atau Sisa Hasil Usaha adalah keuntungan yang diterima oleh anggota koperasi yang dibagikan sesuai dengan aktivitas ekonominya di dalam koperasi tersebut. Untuk cara perhitungannya bisa dilihat dalam pembahasan.

Pembahasan

Besaran SHU yang didapat oleh masing-masing anggota akan beda-beda, karena SHU berasal dari pendapatan anggota dan bukan anggota. Pendapatan anggota terdiri dari jasa usaha seperti penjualan, pembelian atau pinjaman dan jasa modal seperti simpanan. Atau bisa dituliskan :

SHU = Jasa modal + jasa usaha

Jasa modal dan jasa usaha harus dihitung tersendiri dahulu. Yang termasuk dalam jasa modal adalah simpanan wajib dan simpanan pokok, sedangkan simpanan sukarela masuk ke dalam hutang. Yang termasuk dalam jasa usaha adalah penjualan atau pinjaman.

1. Untuk jasa modal Rita :

Jasa modal = (total simpanan Rita/total simpanan koperasi) x persentase jasa modal x SHU

= ((12.000.000 + 1.500.000)/(100.000.000 + 150.000.000)) x 50% x 60.000.000

= 13.500.000/250.000.000 x 30.000.000

= 1.620.000

Untuk jasa usaha Rita :

Jasa usaha = (total penjualan Rita/total penjualan koperasi) x persentase jasa usaha x SHU

= 15.000.000/250.000.000 x 30% x 60.000.000

= 15/250 x 18.000.000

= 1.080.000

Jadi besarnya SHU yang diterima Rita adalah 1.620.000 + 1.080.000 = 2.700.000.

2. Untuk jasa modal Sinta :

Jasa modal = (total simpanan Rita/total simpanan koperasi) x persentase jasa modal x SHU

= ((50.000 + 400.000)/(3.000.000 + 11.400.000)) x 20% x 72.000.000

= 540.000/14.400.000 x 14.400.000

= 540.000

Untuk jasa usaha Sinta :

Jasa usaha = (total pinjaman Sinta/total pinjaman koperasi) x persentase jasa usaha x SHU

= 9.500.000/380.000.000 x 25% x 72.000.000

= 95/38 x 180.000

= 450.000

Jadi besarnya SHU yang diterima Sinta adalah 540.000 + 450.000 = 990.000.

3. Untuk jasa modal Reni :

Jasa modal = (total simpanan Reni/total simpanan koperasi) x persentase jasa modal x SHU

= 1.125/45.000 x 4.500.000

Untuk jasa usaha Reni :
