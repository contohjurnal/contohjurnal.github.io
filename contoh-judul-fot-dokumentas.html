32. ubah kebentuk pasive father drive a car fot two hours​


PART = ACTIVE-PASSIVE VOICE

--------------

ANSWER :

A car is driven (by father) for two hours. ( Passive )

NOTE / DISCUSSION:

"Father drives a car for two hours" merupakan kalimat aktive bentuk simple present. Bisa dicermati pada bentuk kata kerja nya "drives." Kalimat aktive menempatkan si pelaku perkara di posisi utama. Sedangkan kalimat passive menempatkan benda/orang yang dikenakan tindakan aksi di posisi utama kalimat. Maka daripada itu, dalam merubah kalimat aktive ke passive, cermati obyek kalimat nya "a car." Obyek ini akan ditempatkan di awal kalimat , sedangkan subyek ( si pelaku perkara) "father" akan diposisikan sebagai Agent ( diletakkan setelah preposisi "by" ). Selanjutnya terapkan rumus passive voice bentukan simple present tense : Subject (object)+tobe(is/am/are)+verb 3 + by + Agent. Pemilihan tobe , harap diperhatikan bentuk subyek nya , apakah tunggal atau jamak. Kata " a car" merupakan bentuk tunggal, maka penggunaan tobe -nya pake "is"

EXPLANATION:

PASSIVE VOICEPassive voice is focused on the object and the action rather than the doer of the action. When we want to change the active voice to the passive voice, keep in mind that we must put the object at the beginning of sentence as well as the action. Meanwhile, the doer of the action will be the agent in the passive voice.Harap diperhatikan baik-baik, bahwa tidak semua kata kerja yang terlampir dalam kalimat aktif bisa dirubah ke dalam bentuk Passive. Pastikan bahwa kalimat aktif tersebut termasuk transitive verb ( dimana kata kerjanya memerlukan obyek untuk memperjelas informasi ). Passive voice tidak bisa merubah kata kerja tanpa obyek ( Intransitive verb ).  Contoh : Go, sleep, occur, happen .Jika ingin merubah kalimat Aktif ke dalam bentuk Passive Voice, tentukan dahulu subyek dalam kalimat aktif tersebut. Subyek ini akan di posisikan sebagai Agent dalam kalimat pasif. Penerapan By+Agent dapat dihilangkan karena tidak dianggap penting.Selanjutnya, tentukan obyek dalam kalimat aktif . Obyek kalimat aktif akan diposisikan sebagai Subyek dalam kalimat pasif.Harap diperhatikan pronoun nya ( kata ganti orang ). Pronoun akan berubah berdasarkan posisi kelas kata ( parts of speech ) dalam kalimat pasif.Perubahan dalam kalimat passive , menyesuaikan tenses masing-masing , dalam artian tidak merubah informasi yang tertulis pada kalimat aktive nya . Jadi, harap memahami betul bentuk-bentuk tenses dalam bahasa inggris. Ini akan memudahkan dalam mengidentifikasi tenses sekaligus mempermudah dalam perubahan dari kalimat aktif ke bentuk passive.Ciri khas penerapan kalimat passive BE + VERB 3 ( past participle )

Rumus-rumus perubahan dari kalimat aktive ke bentuk passive berdasarkan tenses yang paling sering digunakan   :

1. SIMPLE PRESENT

Subject +base verb (s/es) + Object/complement  ( Active )

Subject  + tobe (is/am/are ) + verb 3 + by + Agent ( Passive )

2. PRESENT CONTINUOUS

Subject + tobe(is/am/are) + verb -ing + object/complement ( Active )

Subject + tobe(is/am/are)+being+verb 3 + by + Agent  ( Passive )

3. PRESENT PERFECT

Subject+has/have+verb3 + object/complement  ( Active )

Subject+has/have+been + verb 3 + by + Agent  ( Passive )

4. SIMPLE PAST

Subject + verb 2 + object / complement  ( Active )

Subject + tobe(was/were) + verb 3 + by + Agent  ( Passive )

5. PAST CONTINUOUS

Subject+tobe (was/were)+verb-ing + object/complement  ( Active )

Subject + tobe (was/were) + being + verb 3 + by + Agent ( Passive )

6. PAST PERFECT

Subject+ had + verb 3 + Object/complement ( Active )

Subject + had + been + verb 3 + by + agent ( Passive )

7. SIMPLE FUTURE

Subject + will + verb1 + Object/complement  ( Active )

Subject + will + be + verb 3 + by + agent  ( Passive )

33. Sebutkan perbedaan deep fot frying, shallow atau panfrying dan vacuum frying


Deep-frying, adalah metode menggoreng dengan minyak jumlah banyak sehingga semua bagian makanan yang digoreng terendam di dalam minyak panas.

Pan frying termasuk teknik memasak dengan menggunakan minyak goreng, tetapi minyak yang digunakan lebih sedikit daripada deep frying.

Shallow frying adalah metode memasak makanan dalam jumlah sedikit, dengan lemak atau minyak yang dipanaskan terlebih dahulu dalam pan dangkal (shallow pan) atau ceper.

Stir frying merupakan metode menggoreng cepat pada temperatur yang sangat tinggi.


34. lost match there were (ten/tenth) fot ballers from manado left


GRAMMAR

Maybe, you mean "Lost match, there were Ten footballers from Manado left". In Indonesian Language, it means "Kalah dalam pertandingan. tersisa 10 pemain dari Manado".

Discussing

The question above is about "Using Cardinal and Ordinal Number". When using cardinal number or ordinal number, there are points that must we must pay attention too.

1. Using Cardinal Number

Cardinal number is used if we only want to say a whole number to state some quantity. Example of cardinal numbers are one (1), two (2), three (3), four (4), five (5), ten (10), twelve (12), etc. These are examples of sentences that used cardinal number ;

Brown has 4 kittens in his house. ⇒ it means that Brown has 4 kittens (1 kitten, 1 kitten, 1 kitten, and 1 more kitten) in his house. It means the quantity of his kittens. On Saturday, I visited 2 supermarkets around the suburban.I and Roza sang 6 songs when We played video tapeIn our galaxy system, there are 8 planets.I don't have any money, could you lend me $4?

- Nouns

Cardinal number is related to Nouns, which means there are several things that we must consider.

If a subject of a sentence is singular, the verb following the noun is in its singular form single verb. Example ;My car is parked in my garage ⇒ car is a singular noun, so we use "is" after the nounThere are many books in my bag ⇒ because the subject is plural, so we use "are". If the subject is a plural noun, we add -s and -es behind the noun. Example :I have two pencilsMy neighbour has two potatoes

2. Using Ordinal Number

Ordinal number is a kind of number that is used to state positions or orders of something series. Example ; first (1ˢᵗ), second (2ⁿᵈ), third (3ʳᵈ), fourth (4ᵗʰ), fifth (5ᵗʰ), twenty-first (21ˢᵗ), thirty-second(32ⁿᵈ), etc. The example :

This is my twentieth (20ᵗʰ) birthday I won the first placeThis is the tenth from umpteenth time of my remedial
